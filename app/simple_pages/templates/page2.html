Importance of AAA Testing

Above is an example of a simple AAA testing using python. We have a function that test if the result of addition equals to two.

1. In line 3, The "Arrange" step creates a variable named num for testing.

2. In line 6, The "Act" step assigns the number 2 to the variable num.

3. In line 9, The "Assert" step verifies that num is equals to two.

The Importance of Testing in Python

Testing serves as an executable documentation or the application, and they also serve as an invaluable reminder of bugs you've fixed in the past. Testing decreases the development time, thus it allows quickly deliver functional code.

  • Detecting problems early - Unit tests discloses problems early into the development.
  • Mitigating change - Allows the developer to refactor the source code during the testing stage and later on, while still making sure the module works as expected.
  • Simplifying integration - By testing the separate components of an application first and then testing them altogether, integration testing becomes much easier.
  • Source of documentation - THe unit testing with doctest provides better insight
